== Grading system

.Your grade consists of these parts:
* `Tsk` -- Tasks solution + code review
* `Cq` -- Colloqium
* `Sem` -- Activity during the semester

NOTE: `max{Tsk} := 90`, `max{Cq} := 10` and `max{Sem} := 10`

Final grade is `min{(Tsk + Cq + Sem)/10, 10}`.
